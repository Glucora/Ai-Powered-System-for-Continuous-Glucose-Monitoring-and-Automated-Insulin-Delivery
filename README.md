# Ai-Powered-System-for-Continuous-Glucose-Monitoring-and-Automated-Insulin-Delivery
Developing a time-series machine learning system to predict blood glucose levels from continuous glucose monitoring (CGM) data for diabetes management. Experimenting with GRU, LSTM, and Hidden Markov Models (HMM) for short-term forecasting, with evaluation metrics focused on clinical relevance. (To be Updated)
